# Design Aesthetics — 12 Named Recipes

Each recipe is a complete starting point: font stack, palette, layout pattern, and signature motion. Mix and match, or use as-is.

## 1. LIQUID MACHINE
**Vibe:** Magazine masthead meets Silicon Valley pitch deck.
- **Fonts:** Bespoke Serif (display) + JetBrains Mono (meta) + General Sans (body)
- **Palette:** `#0a0a0a` bg / `#f5f1ea` text / `#ff3d68` accent
- **Layout:** Two-column 1.1fr/1fr — copy left, video right with mix-blend page number
- **Motion:** Staggered fadeUp serif reveal · scrolling marquee footer
- **Image prompt seed:** "Molten chromatic liquid metal flowing in slow motion, swirling iridescent mercury with pink/cyan/gold reflections, Phase One 150MP, studio lighting, pitch black background, no text"

## 2. NEON PROTOCOL
**Vibe:** Sci-fi product intro for the year 2049.
- **Fonts:** Syne (display) + Space Mono (body)
- **Palette:** `#05050a` bg / `#eaf8ff` text / `#00f0ff` cyan accent / `#ff3caf` magenta secondary
- **Layout:** Full-bleed video with rotating SVG rings + grid overlay + live terminal panel right
- **Motion:** Rotating concentric rings · glitch text on header · blinking cursor · pulse rings
- **Image prompt seed:** "Floating holographic orb of pure neon cyan and magenta light, translucent particles orbiting, volumetric god-rays, Blade Runner 2049 aesthetic, no text"

## 3. BOTANICAL SIGNAL
**Vibe:** Luxury fragrance maison — quiet rooms, gold leaf, deep nature.
- **Fonts:** Cormorant Garamond (display + body italic) + Inter (meta)
- **Palette:** `#060806` bg / `#ebe2c6` cream text / `#d4b874` gold accent
- **Layout:** Centered stack, gold corner-bracket frame, top/bottom meta bars
- **Motion:** Blur-fade reveal on serif title · animated film grain overlay · gradient gold text
- **Image prompt seed:** "Dark moody botanical still life, dewdrops on black monstera leaves, gold-leaf butterfly, Dutch master chiaroscuro, Hasselblad macro, no text"

## 4. DATA MORPHOLOGY
**Vibe:** Nature magazine cover meets scientific report.
- **Fonts:** Fraunces (display + italic captions) + JetBrains Mono (rails + meta)
- **Palette:** `#f4f2ed` paper bg / `#0d0d0d` text / `#cc2b2b` red accent
- **Layout:** Outer 72px rail grid w/ corner plus markers · big number left · video + animated SVG chart right
- **Motion:** Stagger-up digit reveal · animated SVG line chart drawing in · subtle data label tick
- **Image prompt seed:** "Abstract scientific viz, white geometric wireframe bands intersecting iridescent fluid blob on paper-white bg, Swiss design, editorial, no text"

## 5. VAPOR CATHEDRAL
**Vibe:** Risograph creative agency reel — unapologetically weird.
- **Fonts:** Gambarino (display) + Khand (meta) + Supreme (body)
- **Palette:** `#f3d6d6` dusty pink bg / `#1a0e2e` deep plum text / `#ff4d88` hot pink accent / `#fce5a3` butter yellow badge
- **Layout:** Mega kinetic type center w/ mix-blend-difference · spinning circular sticker badge top right · color-wash gradient overlay · grain
- **Motion:** Pop-bounce kinetic letters · rotating SVG textPath badge · wiggling arrow · drifting noise grain
- **Image prompt seed:** "Surreal dreamlike pastel cathedral of vapor, infinite marble columns in pink and cyan mist, oversized moon, Wes Anderson symmetry, James Turrell light, risograph grain, no text"

## 6. BRUTALIST WEB
**Vibe:** Raw concrete. Type as architecture.
- **Fonts:** Helvetica Neue Bold or Inter Black (oversized) + courier mono small caps
- **Palette:** Pure white bg or pure black, single neon accent (yellow / safety orange)
- **Layout:** Type bigger than the viewport allows, intentional overflow, hard horizontal rules, no rounded corners ever
- **Motion:** Sudden pop-in · type-shake on hover · everything snappy 100ms eases
- **Image prompt seed:** "Raw concrete monolith, harsh midday sun, hard shadows, single fluorescent tube glow, brutalist architecture photography, Tadao Ando, no text"

## 7. NEWSPAPER
**Vibe:** New York Times Sunday cover.
- **Fonts:** Fraunces or Old Standard TT (drop cap serif) + Inter for sans accents
- **Palette:** Aged paper `#f5f0e6` / black ink / faded red `#a02525` for masthead
- **Layout:** Multi-column flow with rules between, drop cap, tiny meta lines, footnote style links
- **Motion:** Parallax slow scroll · text crossfade only — no bouncy animations ever
- **Image prompt seed:** "Editorial photojournalism, environmental portrait, Magnum Photos style, available light, 35mm grain, no text"

## 8. GLASSMORPHIC
**Vibe:** Apple keynote / iOS frosted UI.
- **Fonts:** SF Pro Display or Inter (clean modern sans only)
- **Palette:** Soft gradient bg (mesh of `#0066ff` → `#ff00aa` → `#00ffaa`) with all surfaces white-on-blur
- **Layout:** Centered hero card on top of huge looping gradient video, frosted nav, soft drop shadows
- **Motion:** Slow scale on hero · subtle floating cards · gradient mesh slowly shifting
- **Image prompt seed:** "Abstract glass and chrome shapes floating in soft pink-cyan-mint gradient, frosted translucent surfaces, Apple keynote aesthetic, soft volumetric light, no text"

## 9. PRINT POSTER
**Vibe:** Swiss design + Massimo Vignelli + bold ink red.
- **Fonts:** Akzidenz-Grotesk or Neue Haas Grotesk (display + body)
- **Palette:** White bg / black type / single `#ee2e3a` red accent
- **Layout:** Strict 12-column grid · oversized headline + huge negative space · numerical sidebar · single full-bleed image
- **Motion:** None or near-none — let the type breathe. Subtle fadeIn only.
- **Image prompt seed:** "Bauhaus-style geometric composition, primary colors, hard-edged shapes, modernist printmaking, Josef Müller-Brockmann influence, no text"

## 10. ASCII TERMINAL
**Vibe:** Hacker terminal nostalgia / Lain / Matrix.
- **Fonts:** IBM Plex Mono or JetBrains Mono — monospace EVERYTHING
- **Palette:** Black `#000` bg / phosphor green `#00ff41` text / amber `#ffb000` warnings
- **Layout:** ASCII box art borders · simulated CRT scanlines overlay · cursor blink · type-on effect
- **Motion:** Typewriter character reveal · scanline drift · cursor blink · "boot sequence" intro
- **Image prompt seed:** "CRT monitor glow in dark room, green phosphor screen, ASCII art, vintage computer terminal, retro tech aesthetic, no text"

## 11. Y2K CHROME
**Vibe:** 2003 Apple Aqua / liquid metal logos / Razr nostalgia.
- **Fonts:** Eurostile (display) or Magistral + a futuristic mono
- **Palette:** Dark gradient blue → black bg / chrome silver type / icy blue `#7df9ff` glow
- **Layout:** Big chromatic logo center · radial gradients · subtle scanline · floating UI panes
- **Motion:** Slow reflection sweep on chrome · panel slide-ins · cursor pulse
- **Image prompt seed:** "Y2K liquid chrome 3D logo, polished aluminum, blue glass orbs, 2003 Apple Aqua aesthetic, dark gradient background, no text"

## 12. MODERNIST ARCHITECTURE
**Vibe:** Tadao Ando + The Line by Architectural Digest.
- **Fonts:** Times New Roman or Cormorant Garamond italic + a discreet sans for nav
- **Palette:** Concrete `#c4beb6` bg / charcoal `#2a2825` type / one accent of natural wood `#a8794a`
- **Layout:** Wide horizontal split · architectural photography full-bleed · ultra-fine type aligned to grid
- **Motion:** Slow image pan / Ken Burns · type fades in line-by-line · NO bounce, NO spin
- **Image prompt seed:** "Minimalist concrete interior, single ray of light through narrow opening, polished concrete floor, monastic, Tadao Ando, Hasselblad medium format, no text"
