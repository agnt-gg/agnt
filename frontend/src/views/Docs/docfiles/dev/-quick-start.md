## 1. üîë Setting Up Your AI Providers

First, you'll need to get API keys for the AI services you want to use:

### OpenAI API Key
1. Go to [OpenAI's platform](https://platform.openai.com/api-keys)
2. Sign up or log in
3. Click "Create new secret key"
4. Copy your API key

### Anthropic (Claude) API Key
1. Visit [Anthropic's console](https://console.anthropic.com/)
2. Sign up or log in
3. Navigate to "API Keys"
4. Create and copy your key

### Adding Your Keys to AGNT.gg
1. Visit the [Settings](/settings) page
2. Go to the "App Integrations" section
3. Click the button for OpenAI, Anthropic, etc.
4. Paste your keys in the popup field and click "Ok" to save

## 2. üîå Adding Integrations

Connect your favorite apps to supercharge your workflows:

1. Visit the [Settings](/settings) page
2. Go to the "App Integrations" tab
3. Click on the apps you want to connect (like Gmail, Slack, etc.)
4. Follow the prompts to authorize each integration

## 3. üé® Creating Your First Workflow

Let's create a simple workflow:

1. Visit the [Workflow Designer](/workflow-designer) page
2. Drag nodes from the left sidebar onto the canvas
3. Connect nodes by dragging from one node's output to another node's input
4. Click on each node to configure its settings
5. Save and run your workflow

Create custom AI tools to use in your workflows:

1. Visit the [Tool Forge](/tool-forge) page
2. Give your tool a name
3. Write instructions for what you want the AI to do
4. Choose an AI model (e.g., GPT-4 or Claude)
5. Click "Save"

## 5. ‚ö° Adding Custom Tools to Your Workflow

1. Go back to your workflow
